# Resources

> 본문에서 참조하는 자료들의 보관소.

## 폴더 구조 제안

```
resources/
├── images/        ← 다이어그램, 스크린샷
├── diagrams/      ← Mermaid/PlantUML/Excalidraw 소스
├── books.md       ← 분야별 추천 도서
├── courses.md     ← 강의/유튜브 채널
└── links.md       ← 참고 링크 모음
```

## 작성 규칙

- **이미지**는 가급적 원본 + 출처 표기
- **다이어그램**은 가능하면 Mermaid 등 텍스트 기반으로 (Git 친화적)
- **링크**는 죽지 않는 출처 우선 (공식 문서, 학술 자료, GitHub)
