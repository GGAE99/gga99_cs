## Q. Docker란 무엇이며, 개발 환경에서 어떻게 활용하는가?

??? note "정답 보기 ▼"
    - [x] Namespace + cgroup 기반 컨테이너 런타임. 이미지로 재현 가능한 환경 제공
    - [ ] 하드웨어를 가상화해 게스트 OS를 통째로 띄우는 하이퍼바이저
    - [ ] 다중 서버에 컨테이너를 스케줄링·자동 복구하는 오케스트레이션 플랫폼
    - [ ] 코드 의존성을 zip으로 묶어 서버에 업로드하는 패키지 매니저
