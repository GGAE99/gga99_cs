## Q. 1011(2)의 보수값을 구하시오.

??? note "정답 보기 ▼"
    - [x] 1의 보수: 0100, 2의 보수: 0101 (4비트 가정, 1's complement는 비트 반전, 2's complement는 +1)
    - [ ] 1의 보수: 0101, 2의 보수: 0100 (1의 보수와 2의 보수를 서로 바꾼 결과로 흔한 혼동)
    - [ ] 1의 보수: 1100, 2의 보수: 1101 (MSB를 부호 비트로 본 8비트 가정의 잘못된 계산)

> _1의 보수: 0100. 2의 보수: 0101. 비트 수가 명시되지 않은 경우 가정 명확히._

## Q. CPU의 시간 지역성과 공간 지역성에 대해 설명하시오.

    - [ ] 자주 사용되는 데이터는 상위 캐시에, 드물게 쓰이는 데이터는 메인 메모리에 두는 캐시 계층(hierarchy) 배치 전략
