## Q. ML과 LLM에 대해 간략하게 설명하시오.

??? note "정답 보기 ▼"
    - [x] ML은 데이터에서 패턴을 학습하는 알고리즘 군, LLM은 대규모 텍스트로 학습된 Transformer 언어 모델 (ML의 하위)
    - [ ] 딥러닝은 다층 신경망으로 표현 학습을 수행하는 ML의 하위 분야, 신경망은 그 기본 구성 단위인 가중치-활성화 그래프 모델
    - [ ] RAG는 외부 지식 검색을 LLM 추론에 결합하는 패턴, Fine-tuning은 사전학습 모델을 도메인 데이터로 추가 학습시키는 기법
    - [ ] Transformer는 Self-Attention으로 시퀀스를 병렬 처리하는 아키텍처, GPT/BERT 등 현대 LLM의 공통 기반

### 핵심 답안 (30초)
> _ML: 데이터에서 패턴을 학습하는 알고리즘 군. LLM: 대규모 텍스트 코퍼스로 학습된 Transformer 기반 언어 모델 (ML의 하위 분야)._

### 꼬리 질문 후보
- 지도/비지도/강화학습의 차이는?
- LLM의 한계(환각, 컨텍스트 길이)는 무엇이고 어떻게 보완하나? (RAG, Fine-tuning)
- Transformer 구조의 핵심 아이디어는? (Attention)
